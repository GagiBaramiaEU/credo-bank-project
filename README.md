## Structure
- `ge.credo.base` - Test base setup
- `ge.credo.pages` - Page objects

- `ge.credo.util` - Utils (driver, waits, screenshots)
- `ge.credo.data` - Test data and enums
- `ge.credo.assertions` - Custom assertions with screenshots
